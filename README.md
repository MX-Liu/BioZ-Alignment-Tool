# BioZ Alignment Tool

Browser-based alignment and labeling tools for synchronized BioZ, respiration strap, and audio data.

## Contents

- `bioz-aligner.html`: align BioZ, respiration, and audio on a shared timeline, then export shifted or selected segments.
- `label-editor.html`: boundary/label editing workflow.
- `index.html`: dashboard entry page.

## Usage

Open `index.html` in a modern browser, then load synchronized CSV files and optional audio files from the local machine.

The BioZ aligner supports:

- BioZ and audio time shifting
- linked overview/detail plots
- segment selection on the chart
- export of shifted CSV data
- export of selected CSV and audio clips
