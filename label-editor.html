Quick guide

Boundary correction
  1. Load the aligned CSV with existing activity_label values.
  2. Select one label boundary from the editor list.
  3. Drag the vertical line in the plot to shift the boundary earlier or later.
  4. Use nudges for fine correction.
  5. Export the corrected aligned CSV.
