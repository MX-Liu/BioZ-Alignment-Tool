Strumming speed experiment

Run the speed blocks at 60, 90, 120, and 150 BPM. For each speed, cue the first 20 strums with beeps, then ask the participant to continue with 20 additional strums without sound cues before clicking Next.

Protocol

60 BPM 20 cued + 20 silent

Start with 20 beep-guided strums, then 20 more without sound cues.

90 BPM 20 cued + 20 silent

Repeat the same pattern at 90 BPM.

120 BPM 20 cued + 20 silent

Repeat the same pattern at 120 BPM.

150 BPM 20 cued + 20 silent

Finish with the fastest block at 150 BPM.

REST 10 s
